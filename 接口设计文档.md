# 高校外聘老师管理系统 - 接口设计文档

## 一、接口规范

### 1.1 基础信息
- **Base URL**: `http://localhost:8080/api`
- **数据格式**: JSON
- **字符编码**: UTF-8
- **认证方式**: JWT Token

### 1.2 请求头
```
Authorization: Bearer <token>
Content-Type: application/json
```

### 1.3 统一响应格式
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {},
  "timestamp": 1703500000000
}
```

### 1.4 状态码说明
| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或Token过期 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |

## 三、外聘老师管理

### 3.1 分页查询外聘老师列表
```
GET /teacher/page
```

**请求参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| pageNum | int | 否 | 页码，默认1 |
| pageSize | int | 否 | 每页条数，默认10 |
| name | string | 否 | 姓名(模糊查询) |
| phone | string | 否 | 手机号 |
| teacherStatus | int | 否 | 状态(0待聘用 1聘用中 2待续聘 3已解聘) |
| collegeId | long | 否 | 学院ID |
| titleLevel | int | 否 | 职称等级 |

**响应示例**
```json
{
  "code": 200,
  "data": {
    "records": [
      {
        "id": 1,
        "teacherNo": "WT202401001",
        "name": "张三",
        "gender": 1,
        "phone": "13800138001",
        "education": "博士研究生",
        "title": "教授",
        "titleLevel": 4,
        "teacherStatus": 1,
        "collegeName": "计算机学院"
      }
    ],
    "total": 100,
    "size": 10,
    "current": 1,
    "pages": 10
  }
}
```

### 3.2 获取外聘老师详情
```
GET /teacher/{id}
```

### 3.3 新增外聘老师
